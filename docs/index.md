## Quick Start

### Download

Get the latest `.drmlgunit` files from the [GitHub Releases](https://github.com/cldmnky/drumlogue-units/releases) page.

### Installation

1. Connect your drumlogue to your computer via USB
2. Copy `.drmlgunit` files to the appropriate folder:
   - **Synths** â†’ `Units/Synth/`
   - **Reverb FX** â†’ `Units/RevFX/`
3. Safely eject and restart your drumlogue
4. Find the units in the respective slot selections
